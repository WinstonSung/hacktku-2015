HackTKU 2015 Official Website

## Setup

You have to install [Node.js] before getting started.

``` bash
$ npm install hexo-cli -g
$ npm install gulp-cli -g
$ npm install
```

## Server

``` bash
$ hexo server
```

## Build

``` bash
$ hexo generate
$ gulp 
```
